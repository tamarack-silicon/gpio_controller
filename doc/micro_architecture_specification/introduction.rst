************
Introduction
************

Tamarack Silicon Project General Purpose Input/Output controller is a standard GPIO controller, suitable for integration on micro-controllers and SoC.

Features
========

- Support up to 1024 IO pins
- Output enable control via CSR
- Support trigger interrupt on input level change(rising and falling edge)
- Interrupt enable/disable control via CSR

Standards
=========

- AMBA APB Protocol Specification, Version E - ARM `Link <https://developer.arm.com/documentation/ihi0024/e/>`_
